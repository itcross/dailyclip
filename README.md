# DailyClip [CrossIT WebApp Project]

나중에 참고할 링크를 따로 저장했다가 잊어버린적이 있나요?<br>
나를 위한 링크를 폴더로 구분하고 즐겨찾기에 등록해주세요!<br>
나중에 보려고 했는데 잊어버렸다면, Push 기능이 준비되어 있습니다.<br><br>
